#使用说明-V1.2.0

 #Python 版本

  1. 安装依赖：pip install numpy pypng pillow nbtlib
  2. 运行：python SunPixel.py

 #本地 Web 版本

   1. 安装依赖：pip install flask numpy pypng pillow nbtlib
   2. 运行：python SunPixelWeb.py
   3. 在浏览器中访问：http://127.0.0.1:5000
   
 #网络 Web 版本
   1. 在浏览器中访问：https://SunPixel.520cti.shop #不定时开放，加入公告群，获取最新消息[1058766084]

 #Go 版本［暂不可用］

   1. 安装依赖：go mod init SunPixel && go get github.com/MachineMC/NBT
   2. 运行：go build .
   
 #Go 二进制文件版本［暂不可用］
   1. 运行：./SunPixelTool-Android-aarch64
